# HeatSafe HR

AI platforma za rano upozoravanje na toplinske valove i procjenu toplinskog stresa u hrvatskim gradovima, namijenjena gradovima, javnim službama i turističkom sektoru.

## Goal
Build an MVP by 30.04.2026 for Student DIGI Award 2026, with later upgrade potential for Dean's Award and Rector's Award applications.

## Core idea
HeatSafe HR combines weather data, heat-risk logic, and machine learning to estimate and visualize heatwave risk across Croatian cities.

## Cities in v1
- Zagreb
- Split
- Rijeka
- Osijek
- Zadar
- Dubrovnik
- Šibenik

## Planned modules
- data ingestion
- preprocessing
- feature engineering
- risk engine
- machine learning model
